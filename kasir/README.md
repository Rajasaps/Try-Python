# Kedai Fauget - Aplikasi Pemesanan Makanan

Website aplikasi pemesanan makanan dengan tampilan modern dan responsif.

## Instalasi

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Jalankan aplikasi:
```bash
python app.py
```

3. Buka browser dan akses:
```
http://localhost:5000
```

## Fitur

- Dashboard dengan grid menu makanan
- Sidebar navigasi
- Profil pengguna
- Daftar pesanan terkini
- Pencarian menu
- Multi-bahasa (Indonesia/English)

## Struktur Project

- `app.py` - Aplikasi Flask utama
- `templates/index.html` - Template HTML
- `static/style.css` - Styling CSS
- `requirements.txt` - Dependencies Python
